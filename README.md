Telegram Points Bot

ربات تلگرام ساخته شده با Python و pyTelegramBotAPI

امکانات

- سیستم امتیاز
- لیدربورد
- درخواست امتیاز
- پنل ادمین
- ذخیره اطلاعات در JSON
- سازگار با Render

---

نصب

1. کلون پروژه

git clone YOUR_REPO_URL

2. نصب وابستگی‌ها

pip install -r requirements.txt

3. تنظیم Environment Variable

BOT_TOKEN=YOUR_BOT_TOKEN

4. اجرا

python main.py
